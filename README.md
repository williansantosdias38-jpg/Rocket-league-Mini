# Rocket-league-Mini
Linguagem de programação: Python
IDE utilizada:Visual Studio Code

Foi utilizada a biblioteca Pygame Zero (pgzrun), que facilita a criação de jogos em Python.
Também usa recursos do Pygame, que é uma biblioteca muito conhecida para desenvolver jogos 2D.

Resumo do funcionamento:

O jogo é um mini Rocket League, onde carros tentam empurrar uma bola para fazer gols.
O código controla os movimentos dos carros, a movimentação da bola, a contagem de gols, os menus e os efeitos sonoros.
Como controlar o jogo

No código, os controles são:

W = andar para cima
A = andar para a esquerda
S = andar para baixo
D = andar para a direita

Também funcionam as setas do teclado:

↑ = cima
↓ = baixo
← = esquerda
→ = direita
